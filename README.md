# Projekt: SmartWeb Platform

## Glavne funkcionalnosti
1. **Uporabniška prijava in registracija** – omogoča prijavo z e‐pošto ali družbenimi omrežji.
2. **Nadzorna plošča** – prikaz statistike, poročil in obvestil.
3. **Sistem za upravljanje vsebine (CMS)** – uporabniki lahko dodajajo, urejajo in brišejo vsebine.
4. **Integracija plačilnega sistema** – podpora za kreditne kartice, PayPal ipd.
5. **Napredna iskalna funkcija** – hitro iskanje elementov po kategorijah in filtrih.
